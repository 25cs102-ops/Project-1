# Java Calculator

A simple console-based calculator application developed in Java.  
The program performs basic arithmetic operations based on user input.

---

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero

---

## Technologies Used
- Java
- Online Compiler (Replit / Programiz)

---

## How to Run

1. Open an online Java compiler such as Replit or Programiz  
2. Copy and paste the code into the editor  
3. Click the "Run" button  
4. Enter input values when prompted  
5. View the result in the output console
